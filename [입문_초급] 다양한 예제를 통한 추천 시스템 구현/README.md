# [입문·초급] 다양한 예제를 통한 추천 시스템 구현 — 강의 소개 정리

---

## 강의 목표

- 복잡한 수식/이론 중심 설명보다는 **직접 구현**하면서 추천시스템 핵심 개념을 익힌다.
- 대상: **초급자**
- 총 **12개**의 다양하고 실용적인 예제로 구성
- **콘텐츠 기반 추천**, **협업 필터링**, **딥러닝 추천**까지 실제 환경에서 활용 가능한 추천 시스템 설계를 다룬다.
- **Surprise**, **LightFM** 등 검증된 추천 라이브러리를 사용한다.

---

## 강의 사전 지식(권장)

- **Python**
  - pandas, scikit-learn, pytorch 등 실습에 필요한 라이브러리를 기본적으로 사용할 수 있으면 좋다.
- **Pandas**
  - 데이터 처리의 핵심 도구
  - 추천 모델링 전에 **데이터 패턴을 파악**하는 데 필수
- **Google Drive + Colab**
  - 클라우드 기반 실습 환경
  - 무료 GPU 지원(필요 시)

---

## 강의 구성 순서

- **기초(통계) → 응용(머신러닝) → 고급(딥러닝)**

### 큰 흐름(요약)

1. **통계 기반 추천**
   - 인기 기반
   - 조회수 기반
   - 평점 기반
2. **머신러닝 기반 추천**
   - 콘텐츠 기반
   - 협업 필터링
3. **딥러닝 기반 추천**
   - 임베딩
   - 트랜스포머
   - 시퀀스 모델
